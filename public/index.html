Task Management API

API untuk menyimpan task secara online. Digunakan untuk project Task Management App.

Endpoint

http://localhost:8000/api

https://task-management-api-liard.vercel.app/api

User

Register

  • URL:
    • /register
  • Method:
    • POST
  • Request Body:
    • name as string
    • email as string, must be unique
    • password as string, must be at least 6 characters
  • Response: 
                {
              "status": "success",
              "message": "User Created"
            }

Login

  • URL:
    • /login
  • Method:
    • POST
  • Request Body:
    • email as string
    • password as string
  • Response: 
                {
              "status": "success",
              "message": "User logged successfully",
              "data": {
                  "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VySWQiOiJ1c2VyLXlqNXBjX0xBUkNfQWdLNjEiLCJpYXQiOjE2NDE3OTk5NDl9.flEMaQ7zsdYkxuyGbiXjEDXO8kuDTcI__3UjCwt6R_I"
              }
          }

Get User logged in

  • URL:
    • /login
  • Method:
    • GET
  • Headers:
    • Authorization : Bearer {accessToken}
  • Response: 
                {
              "status": "success",
              "message": "User retrieved",
              "data": {
                  "id": "user-yj5pc_LARC_AgK61",
                  "name": "John Doe",
                  "email": "john@example.com"
              }
          }

Tasks

    • /tasks
  • Method:
    • POST
  • Headers:
    • Authorization : Bearer {accessToken}
  • Request Body:

              }
            }

Archive Task

  • URL:
    • /tasks/{task_id}/archive
  • Method:
    • POST
  • Headers:
    • Authorization : Bearer {accessToken}
  • Response: 
                {
              "status": "success",
              "message": "Task archived"
            }

  • Method:
    • POST
  • Headers:
    • Authorization : Bearer {accessToken}

Delete Task

  • URL:
    • /tasks/{task_id}
  • Method:
    • DELETE
  • Headers:
    • Authorization : Bearer {accessToken}
  • Response: 
                {
              "status": "success",
              "message": "Task deleted"
            }

Update Task

  • URL:
    • /tasks/{task_id}
  • Method:
    • PUT
  • Headers:
    • Authorization : Bearer {accessToken}
  • Request Body:
    • title as string
    • body as string
    • archived as boolean
    • status as string
  • Response: 
                { 
              "status": "success",
              "message": "Task updated",
              "data": {
                  "id": "67611fc1836036d937cfe139",
                  "title": "Amazing Task now a day",

                  "createdAt": "2024-12-17T06:52:49.959Z",
                  "archived": true,
                  "owner": "67611f5a836036d937cfe135"

              }
